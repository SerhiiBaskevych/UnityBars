Вітаю!
Створив метод з вказаною сигнатурою.
Створив два набори даних з дозволяючими та забороняючими умовами, як вказано в примітці, враховуючи аут пут, який вказаний в файлі
Не зромуів до кінця формулювання в примітці "в одному масиві даних можуть бути тільки дозволяючі умови, або тільки забороняючі."
Як зрозумів?:
Flag_NBS, Flag_OB22 - умови, які валідуються забороняючі/дозволяючі як 0/1.
тобто два масива можуть зайти в метод в яких будуть флаги Flag_NBS, Flag_OB22 == 1, або == 0, комбінація в одному масиві Flag_NBS == 1, Flag_OB22 == 0
Не можлива якщо правильно зрозумів.
Але це не стекається з прикладом вхідних параметрів в файлі де Flag_NBS: 1, Flag_OB22:0, тобто в одному масиві є і забороняюча і дозволяюча умова.
Як зробив?:
Згідно таблиці з дозволами.
id 1, 2 - з прикладу вхідних параметрів (в таблиці 1,2 рядок як зрозумів)
id 3, 4 - останні два записи з таблиці.
Дякую.
